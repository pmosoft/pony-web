Make Easy

Many developers find it difficult to create web diagrams. Can not we make this easily? Especially HMI, Start Factory.

The goal is to make a diagram editor so that anyone can easily create a Smart Factory Visualization.


Editor based

Make diagram by coding? Arn't you difficult?

Your can make features in Editor (similar to Powerpoint eidtor). include position, rotate, skew, fill, stroke, path move, ajax params etc.


Rapid deploy

If the Diagram is modified, it will be difficult to do the deployment again. Would it be easy to be reflected right away?

It can be used anywhere on the web browser and loaded with simple code. It is located on the same web service that is provided and is ready to use without any deployment work.


Component based

Using A component, A 'component is needed. You have to create a new one? Need a developer? no!

You can add similar components or create new components and use them right away with simple modifications without developing the component each time.
